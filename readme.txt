A beautiful, floating table of contents block that automatically generates navigation from your page headings.

== Description ==

NavyGator is a modern, user-friendly table of contents block for WordPress that automatically generates a navigation menu from the headings in your content. It features a sleek floating design on desktop and a mobile-friendly toggle drawer.

= Key Features =

* **Automatic Generation**: Automatically extracts headings from your content
* **Customizable Heading Levels**: Choose which heading levels (H1-H6) to include
* **Desktop Floating View**: Beautiful floating sidebar that stays visible while scrolling
* **Mobile-Friendly**: Toggle button with slide-up drawer for mobile devices
* **Scroll Spy**: Automatically highlights the current section as you scroll
* **Smooth Scrolling**: Smooth scroll animation when clicking TOC links
* **Customizable Colors**: Set custom background and text colors
* **Numbered or Bulleted**: Choose between numbered or bulleted lists
* **Custom Title**: Set a custom title for your table of contents
* **Nested Headings**: Properly handles nested heading structures
* **Accessible**: Built with accessibility in mind (ARIA labels, keyboard navigation)
* **Lightweight**: Minimal performance impact

= Perfect For =

* Long-form blog posts
* Documentation pages
* Tutorials and guides
* Academic articles
* Product descriptions
* Any content with multiple sections

= How It Works =

1. Add the NavyGator block to your post or page
2. Customize the settings (heading levels, colors, title)
3. The table of contents is automatically generated from your content
4. On desktop, it floats on the right side of the screen
5. On mobile, users can toggle it with a button

= Technical Details =

* Uses modern WordPress Block API (API version 3)
* Dynamic rendering for optimal performance
* Intersection Observer API for scroll spy
* CSS Grid and Flexbox for responsive layouts
* No jQuery dependency
* Compatible with Full Site Editing (FSE)

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to Plugins > Add New
3. Search for "NavyGator"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin ZIP file
2. Log in to your WordPress admin panel
3. Navigate to Plugins > Add New > Upload Plugin
4. Choose the ZIP file and click "Install Now"
5. Activate the plugin

= Usage =

1. Create or edit a post/page
2. Click the (+) button to add a new block
3. Search for "NavyGator" or "Table of Contents"
4. Add the block to your content
5. Customize settings in the block sidebar
6. Publish your post/page

== Frequently Asked Questions ==

= Does this work with any WordPress theme? =

Yes! NavyGator works with any WordPress theme that supports the block editor (Gutenberg).

= Can I customize the appearance? =

Yes, you can customize the background color, text color, and choose between numbered or bulleted lists. Additional styling can be added via custom CSS.

= Does it work on mobile devices? =

Absolutely! On mobile devices, the table of contents is hidden by default and can be accessed via a floating toggle button that opens a slide-up drawer.

= What heading levels are included by default? =

By default, H2, H3, and H4 headings are included. You can customize this in the block settings to include any combination of H1-H6.

= Does it affect page load speed? =

NavyGator is designed to be lightweight and has minimal impact on page load speed. Scripts and styles are only loaded on pages that actually use the block.

= Can I use multiple TOC blocks on one page? =

The block is designed to be used once per page for the best user experience. Multiple instances may cause conflicts.

= Does it work with custom post types? =

Yes, NavyGator works with any post type that supports the block editor.

= Is it accessible? =

Yes, NavyGator is built with accessibility in mind, including proper ARIA labels, keyboard navigation support, and semantic HTML.

= Does it work with page builders? =

NavyGator works best with the native WordPress block editor. Compatibility with third-party page builders may vary.
